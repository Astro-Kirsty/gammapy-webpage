The Gammapy project

Gammapy is a community-developed, open-source Python package for gamma-ray astronomy. It is an in-development affiliated package of Astropy that builds on the core scientific Python stack to provide tools to simulate and analyse the gamma-ray sky for telescopes such as CTA, H.E.S.S., VERITAS, MAGIC, HAWC and Fermi-LAT. Gammapy is a place for Python-coding gamma-ray astronomers to share their code and collaborate. Feature requests and contributions welcome!
