Cherenkov Telescope Array

Introduction

The Cherenkov Telescope Array is the next generation ground-based observatory for gamma-ray astronomy at very-high energies. Gammapy is a prototype for the CTA science tools (see 2017arXiv170901751D).

This page provides a little bit of information and links to useful resources concerning simulation and analysis of CTA data with Gammapy. Most of the pages we link to here require a CTA user account to access the information.

Getting started

To learn how to use Gammapy for CTA analysis, use the Gammapy tutorials. We suggest you do the "Getting started with Gammapy" one first, then the "CTA first data challenge (1DC) with Gammapy" and then the "CTA data analysis with Gammapy".

If you have questions, please contact the gammapy-cta-l mailing list or the gammapy-coordination-l mailing list (info)

Please note that Gammapy is a very young project and is under heavy development. At the moment we are participating in the CTA first data challenge, fixing issues and adding new functionality for CTA.
