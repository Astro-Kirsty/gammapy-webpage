Getting Help with Gammapy

  1. Gammapy Slack team (get an account here)

    Most of the developers and contributors are using Slack as chatting tools. Post any question you have and you will receive an answer within some hours, depending on the time of the day.

  2. User Email List (join it here)

    This is a free mailing list. Ask any question about Python programming applied to gamma-ray astronomy.

  3. CTA User Email List

    Ask any question about C.T.A. data analysis with gammapy.

  4. Coordination Committee Email List

    Post any question about gammapy visibility
