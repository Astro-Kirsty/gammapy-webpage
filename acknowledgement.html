Acknowledging Gammapy

  1. In Publications

    If you use Gammapy for work/research in a publication, we ask that you cite the Gammapy proceeding (ADS - BibTeX). We provide the following as a standard acknowledgment you can use if there is not a specific place to cite the paper: This research made use of Gammapy, a community-developed core Python package for gamma-ray astronomy 2017arXiv170901751D> (Deil et al. 2017)

  2. In Presentations

    This is a free mailing list. Ask any question about Python programming applied to gamma-ray astronomy.

  3. In Projects

    Ask any question about C.T.A. data analysis with gammapy.
